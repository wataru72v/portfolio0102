サービス開発

自分の作品を世に発表したいという思いから、WEBサービスの開発を行っています。サービスを開発しながら、技術のキャッチアップ、情報のアウトプットなども行っていきます。

動画投稿

開発の様子や有益な情報を動画やライブ配信を通して共有します。自分のモチベーションを上げ、同時に視聴者のモチベーションを上げられるようなコンテンツを目指します。

学習教材

活動の中で得られた知見や情報は学習教材に落とし込み販売します。ここで得た資金はサービスの開発/運用費、動画や次の学習教材の質を向上させるために活用します。
